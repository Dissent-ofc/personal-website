# Personal Website

This is my personal website, built with Webpack and deployed on Vercel.

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Deployment (Vercel)

This repository is configured for Vercel via `vercel.json`:

- Build command: `npm run build`
- Output directory: `dist`

If you configure settings manually in Vercel dashboard, use the same values above.
